Module latexplots

source code

Draws publication-quality plots for use in the paper

The figures are: 
   Figure 1. Four score density plots
   Figure 2. ROC curve overlay
   Figure 3. PR curve overlay
   Figure 4. P,R,F1,Fa curve for AIDSBio to demo optimisation
