Class FileTransaction

            FileTransaction

Transaction for Cheetah templates to output direct-to-file.

Cheetah defaults to DummyTransaction which creates a huge list and joins them up to create a string. This is way slower than writing to file directly.

Usage: 
   with FileTransaction("something.html","wb") as ft:
       Template().respond(ft)
