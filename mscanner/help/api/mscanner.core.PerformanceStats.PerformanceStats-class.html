make_curve_areas(self)

source code 

Calculate areas under ROC and precision-recall curves

Uses trapz(y, x). TPR is decreasing as threshold climbs, so vectors have to be reversed.

This method underestimates ROC areas because boundary points (0,0) and (1,1) usually are not present in the data. Better to use roc_error which does not have that problem.
Returns:
ROC_area, PR_area

roc_error(self)

source code 

Area under ROC and its standard error

Uses method of Hanley1982 to calculate standard error on the Wilcoxon statistic W, which corresponds to the area under the ROC by trapezoidal rule.
Returns:
W, W_stderr

Note: The vectors r1 .. r7 correspond to rows of Table II in Hanley1982.
