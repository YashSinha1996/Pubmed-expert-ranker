Class Plotter

source code

Known Subclasses:
DensityPlotter

Implements the plots used in MScanner

When adding a new analysis, the plotting function for its output graphs should be added here.

Note: All methods take an fname parameter, which is the path to the PNG file to which the graph will be written.
